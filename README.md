# Backend Test Case

## Usage Installation

- Clone Repo

```shell
git clone https://github.com/muhammadisa-n/backend-test-case.git
```

- Create .env file

```shell
cp .env.example .env
```

- Run npm install

```shell
npm install
```

- Run prisma

```shell
npx prisma migrate dev
```

```shell
npx prisma generate
```

- If you want to seeding database

```shell
npm run seed
```

- Run Server

```shell
npm start
```

Or

```shell
npx run dev
```

## URL

- BASE_URL = http://localhost:3000/api
- Access Swagger Documentation = http://localhost:3000/api-docs
